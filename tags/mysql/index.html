Q4Mでのレプリケーション

on 2010-04-21

Q4Mを使用したテーブルにデータをインサートすると

#1598 - Binary logging not possible. Message: Statement cannot be logged to the binary log in row-based nor statement-based format

というエラーが表示される。バイナリログが有効だとインサートにミスるみたい。ということで、バイナリログを使用しないようにする。
