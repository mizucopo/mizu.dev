『Physical Layer』クリア後レビュー

2023-04-01 — 2 min read — Tweet
#日記  #ゲーム  #2Dアクションゲーム  #見下ろし型シューティングゲーム  #RaStar Studio  #Phoenixx  #Physical Layer 

『Physical Layer』は、女子高生がエージェントとして活躍する見下ろし型ガンシューティングアクションゲームです。 プレイヤーは、敵によって起動されたEMP兵器の停止コードを回収することが目的で、ステージごとに様々な地形と敵が待ち受けます。 瞬時な判断が求められ、最適解を導く必要があります。 クリア後の感想を交えて本記事では、クリア後の感想を交えて、ゲームの良い点や悪かった点などを記載していきます。

【良い点】 #

  1. いろんなギミックがあり、それらを理解すると面白くなってくること。
  2. 反転スキルが斑鳩っぽいので、楽しめること。
  3. コスチュームが可愛いこと。

【悪い点】 #

  1. オブジェクト投げが、オブジェクトが複数個ある場合、どのオブジェクトを投げるか選択できないため、やきもきすること。
  2. リトライ時のウェイト時間がないため、すぐに復帰してしまいダメージを受けてしまう場合があること。
  3. ボス専用のBGMがないこと。
  4. （私の環境では）反転スキルを利用するとひっかかりがある場合があること。
  5. （最後までプレイしていないかもしれませんが）エンディングがないこと。

『Physical Layer』は、面白いゲームであり、安価で手軽にプレイできる点が魅力的です。 また、難易度は高くはありませんが、難易度を高めて遊ぶことができます。 改善すべき点もありますが、それでも一度プレイしてみる価値はあるでしょう。
