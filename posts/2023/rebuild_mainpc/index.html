メインPCを新たに組み直しました

2023-08-07 — 2 min read — Tweet
#日記  #パソコン  #自作PC 

久しぶりにメインPCを刷新しました。 『CPU (Ryzen 9 3900X) が壊れました 』というのもあり、刷新した感じになります。 今回はそのメインPCの、性能周りを確認していきます。

メインPCの構成は以下のようになっております。

  • CPU: Intel Core i7 8700K (6Core 12Thread) → Intel Core i7 13700KF (16Core 24Thread)
  • CPU Cooler: noctua NH-L9i → DeepCool LT720
  • Mother: GIGABYTE H370N WIFI → Asrock Z790 LiveMixer
  • Memory: G.skill F4-3600C16-32GVKC (32GB) → ADATA AD5U480016G-BD-JP DDR5-4800 16GB * 4 (64GB)
  • Video: ELSA GeForce RTX 3060 Ti ERAZOR LHR → 玄人志向 GG-RTX4090-E24GB/OC/TP
  • Power Supply: Corsair SF600 (600W) → Corsair RM850e (850W)
  • Case: Fractal Design Torrent → NZXT H9 Flow
  • OS: Windows 11 Pro 21H2 (22000.675) → Windows 11 Pro 22H2 (22621.2070)

AI関連のツールを利用することもあり、かつ、最近のゲームはVRAM8GBというのは心もとないというのもあり、ここは奮発してRTX4090にしてみました。 StableDiffusionを利用してみましたが、やはり画像生成はいままで比べ圧倒的に早いですね。 ゲームはまだ未プレイなので、感動は味わえていないです。サイバーパンクがセール時に購入して試してみます。

ベンチマークのスコアは下記になります。

  • CINEBENCH R23.200
    • CPU (Multi Core) 6619 → 26977
    • CPU (Single Core) 1161 → 2050
  • Fire Strike (v1.1): 22306 → 53533
    • グラフィックスのスコア: 29817 → 77371
    • 物理スコア: 16806 → 42477
    • 総合スコア: 8849 → 18332
  • Fire Strike Extreme (v1.1): 39623
    • グラフィックスのスコア: 45647
    • 物理スコア: 42211
    • 総合スコア: 19034
  • Fire Strike Ultra (v1.1): 24792
    • グラフィックスのスコア: 24949
    • 物理スコア: 43310
    • 総合スコア: 14683
  • Time Spy (v1.2): 9909 → 32306
    • グラフィックスのスコア: 10662 → 36767
    • CPUスコア: 7080 → 19144
  • Time Spy Extreme (v1.2): 17057
    • グラフィックスのスコア: 19466
    • CPUスコア: 10028

スコア的に前利用していたPCと比べ、2〜3倍になっておりますね。 お金をかけただけあって、満足いく結果になっております。
