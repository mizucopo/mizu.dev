CakePHP の shell のウェルカムメッセージを非表示に

2011-04-03 — 1 min read — Twitter
#開発  #CakePHP 
Class WelcomeShell extends Shell {
  function startup() {
  }
}

startup() メソッドをオーバーライドすれば、ウェルカムメッセージが表示されない。

ちなみにウェルカムメッセージは _welcome() メソッド定義されている。

おまけ #

shell での文字列の出力ですが、echo すれば文字列が出力されるが、ここは、out() メソッドを使うのがいいみたい。
また、ラインの出力は hr() メソッド。
