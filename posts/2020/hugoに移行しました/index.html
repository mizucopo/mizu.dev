Hugoに移行しました

on 2020-02-11

  • Twitter
#Hugo  #日記 

WordPress から Hugo に移行しました。 移行したのは、ずっと前で、エントリーするのをめんどくさがってやっておりませんでした。

Hugo に移行した理由は、あまり管理していないブログにお金を支払い続けるのは、もったいないなと思ったのが理由です。

利用している技術構成は下記になります。

  • Hugo
  • Github Pages
  • Github Action

Github にエントリーをプッシュすることで、ページがビルドされ、それを公開する形になります。 できるだけ、手間をかけずにブログを更新できるようにしてます。

Previous Serverless を利用して S3 の総ファイルサイズを Slack へ投稿する Next アイコンを作ってもらった
