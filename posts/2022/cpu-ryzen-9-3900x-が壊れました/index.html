CPU (Ryzen 9 3900X) が壊れました

on 2022-05-26

  • Twitter
#パソコン  #自作PC 

3月末の話になりますが、PCの電源が入らなくなりました。 いろいろ代替パーツを切り替え問題個所を洗い出したところ、マザーボードもしくはCPUのどちらかが異常ということがわかりました。

というわけで、マザーボードおよびCPUを修理に出してきました。

結果からいうと、CPUの故障ということがわかり、返金対応ということになりました。 1世代前のCPUが満額返ってくるということで、ちょいラッキー。65,000円。

次世代のCPUが秋ぐらいにリリースされそうなので、その間、余ったパーツでしばらく運用します。

ちなみに今回の件とは関係ないのですが、PCケースを変更しております。

  • CPU: AMD Ryzen 9 3900X (12Core 24Thread) → Intel Core i7 8700K (6Core 12Thread)
  • CPU Cooler: NH-U12S chromax.black → noctua NH-L9i
  • Mother: B450 I AORUS PRO WIFI → GIGABYTE H370N WIFI
  • Memory: G.skill F4-3600C16-32GVKC (32GB)
  • Video: ELSA GeForce RTX 3060 Ti ERAZOR LHR
  • Power Supply: Corsair SF600 (600W)
  • Case: オープンフレーム ver.mini-ITX → Fractal Design Torrent
  • OS: Windows 10 Pro 21H1 (19043.1237) → Windows 11 Pro 21H2 (22000.675)

  • FIRE STRIKE (v1.1): 22742 → 21800
    • グラフィックスのスコア: 28251 → 28668
    • 物理スコア: 26295 → 17068
    • 総合スコア: 8534 → 9157
  • TIME SPY (v1.2): 11482 → 9754
    • グラフィックスのスコア: 11489 → 10577
    • CPUスコア 11449 → 6770
  • PORT ROYAL (v1.2): 6875 → 6518
    • グラフィックスのスコア 6875 → 6518
  • CINEBENCH R23.200
    • CPU (Multi Core) 17927 → 7119
    • CPU (Single Core) 1273 → 1099

上記結果になりました。3回計測してその中央値をとっております。

やはりCPUが悪くなっておりますね。知りたくはなかった。。。 次のアーキテクチャを待ちます。

ちなみにマザーボードも異常があり、フロントUSB3.0の端子が壊れていた模様です。 修理されて返ってきました。
